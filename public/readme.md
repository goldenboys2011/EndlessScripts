Welcome to the **Endless Scripts Public Edition**!

This is a public script pool that allows anyone to install custom cheat scripts directly through the game.

## Install a Script

In-game, simply run:

```
script install {name}
```

This will automatically download `{name}.enscript`
from the public script pool and add it to your local scripts folder.

- Only upload **safe, single-file scripts** (no external network calls or malicious code).

- You can update your script anytime by submitting a new PR.
